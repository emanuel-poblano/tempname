# FinPath MVP

This workspace contains a lightweight MVP for a bilingual financial wellness platform that combines:

- an AI financial coach experience
- brokerage referral guidance
- a paid investing community model

## Run locally

Open the app in a browser at http://127.0.0.1:8000.

If you need to restart the preview server, run:

```bash
cd /workspaces/tempname
python3 -m http.server 8000
```

## Test

```bash
cd /workspaces/tempname
npm test
```
